# HR Data Automation Tool

Программа на Python для автоматизации сбора данных из разных источников (SQLite).

### Что делает:
* Объединяет данные о проектах и зарплатах сотрудников из разных баз данных.
* Умеет сопоставлять сотрудников, даже если их ФИО написаны по-разному (с точками, в разном регистре и т.д.) благодаря функции нормализации.
* Считает итоговую выплату (оклад + бонус) и сохраняет результат в одну общую базу `warehouse.db`.

### Как протестировать:
1. Установить Python.
2. Запустить файл `main.py`.
3. Программа сама создаст тестовые базы и выведет результат в консоль.
